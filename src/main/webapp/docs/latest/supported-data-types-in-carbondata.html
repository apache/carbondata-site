Data Types

CarbonData supports the following data types:

    Numeric Types
  • SMALLINT
  • INT/INTEGER
  • BIGINT
  • DOUBLE
  • DECIMAL
    Date/Time Types
  • TIMESTAMP
  • DATE
    String Types
  • STRING
  • CHAR
    Complex Types
  • arrays: ARRAY<data_type>
  • structs: STRUCT<col_name : data_type COMMENT col_comment, ...>
