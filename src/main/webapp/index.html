What is CarbonData

Apache CarbonData (incubating) is an indexed columnar data format for fast analytics on big data platform, e.g. Apache Hadoop, Apache Spark, etc.

Join CarbonData Community

Apache CarbonData is an open source project of The Apache Software Foundation (ASF). We are an open and friendly community. We welcome everyone to join the community and contribute to CarbonData.

It is very easy to join CarbonData community:

  • Subscribe for the mailing list by sending an email to dev-subscribe@carbondata.incubator.apache.org
  • After you subscribe, you can discuss your ideas, queries, and issues by sending an email to dev@carbondata.incubator.apache.org or visit http://apache-carbondata-mailing-list-archive.1130556.n5.nabble.com
  • To start contributing to CarbonData and be a contributor, see Contributing to Apache CarbonData.
  • To report issue on Apache Jira.

Why CarbonData

Unique Data Organization

Stores data in Columnar format, with each Data Block(row group) sorted independent of the other to allow faster filtering and better compression.

Multi Level Indexing

Utilizes multiple indices at various levels to enable faster search and speeding up query processing.

Deep Spark Integration

DataFrame & SQL compliance.

Advanced Push Down Optimization

Pushes much of query processing close to the data to minimize the amount of data being read, processed, converted, transmitted and shuffled.

Dictionary Encoding

Encoded data for reduced storage space & faster processing.

Update & Delete Support

Support update and delete over BigData.
